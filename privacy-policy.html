プライバシーポリシー

最終更新日: 2026年5月10日

本ポリシーは、ppoonn（以下「当方」）が提供するモバイルアプリケーション「MoonNote（ムーンノート）」（以下「本アプリ」）における、ユーザーの情報の取り扱いについて定めるものです。

1. 取得する情報

本アプリは、以下の情報を必要に応じて取得します。

  • 位置情報：現在地の月の出入り時刻と天気を取得するために使用します。OS の位置情報許可がある場合のみ取得し、許可されていない場合や手動で都市を選択している場合は、ユーザーが選んだ都市の座標を使用します。
  • 日記本文・気分タグ・月齢・天気情報：ユーザーが入力した日記本文、選択した気分タグ、その日時点の月齢・月相・天気情報。これらは原則として端末内のローカルストレージにのみ保存されます。
  • アプリの設定情報：通知のオン／オフ、リマインダー時刻、表示設定、選択言語など。端末内に保存されます。

2. 第三者サービスへの送信

本アプリは、以下の機能を提供するために、必要最小限の情報を外部サービスに送信します。

  • OpenWeatherMap（天気情報の取得）：緯度経度を OpenWeatherMap API に送信し、現在地の天気情報を取得します。日記本文や個人を特定する情報は送信しません。
  • Google Gemini API（「今夜の言葉」生成）：ユーザーが入力した日記本文、気分タグ、月齢・月相、天気情報を Google の Gemini API に送信し、AI が生成する短い言葉を取得します。送信内容にはユーザーを直接特定する情報（氏名・メールアドレス・端末識別子等）は含まれません。Google による Gemini API の取り扱いについては、Gemini API 追加利用規約および Google プライバシーポリシーをご確認ください。

これらの送信は、当該機能を利用するときにのみ行われます。送信されたデータは当方のサーバーには保存されません。

3. 端末内に保存される情報

本アプリは原則として、ユーザーのデータを端末外のサーバーに保存しません。日記本文・気分タグ・設定情報は、すべて端末内のローカルストレージ（Hive / SharedPreferences）に保存されます。

端末をバックアップ（iCloud バックアップ・Google ドライブのバックアップ等）した場合、OS のバックアップ機能に従ってクラウドにバックアップされる場合があります。これらは OS 提供元（Apple / Google）が管理するものであり、当方はアクセスできません。

4. 通知

本アプリの通知（日記リマインダー、観月通知等）は、すべて端末上で動作するローカル通知です。プッシュ通知サーバーは利用しておらず、当方が通知の内容や送信先を取得することはありません。

5. アナリティクス・広告

本アプリは、ユーザーの行動を追跡するアナリティクス SDK や広告 SDK を組み込んでいません。クラッシュレポートや使用状況の解析のための SDK も使用していません。

6. 課金情報

プレミアム機能の購入は、Apple App Store または Google Play ストアの決済システムを通じて行われます。クレジットカード番号などの決済情報は Apple / Google が管理し、当方が取得することはありません。購入の有効性確認には、ストアが発行するレシート情報のみを利用します。

7. 子どもの利用

本アプリは全年齢を対象としていますが、13 歳未満のユーザーから個人を特定できる情報を意図的に収集することはありません。

8. ユーザーの権利

ユーザーは以下の操作をいつでも行うことができます。

  • 設定画面から、日記データをエクスポート／インポートする
  • 設定画面の「全データを削除」から、端末内のすべての日記・設定・キャッシュを完全に削除する
  • OS の設定から、位置情報・通知の権限を取り消す

9. ポリシーの変更

本ポリシーの内容は、必要に応じて変更されることがあります。重要な変更がある場合、本アプリまたはストア掲載情報を通じてお知らせします。本ポリシーの変更は、本ページに掲載した時点で効力を生じるものとします。
